Project overview
Made a bike renting system where users can log in to screens attached to bikes using username and password. They can use and then drop the bikes at designated points. The project is based on microservices architecture where individual services handle specific functionalities like authentication, billing, inventory management and billing etc. PostgreSQL is used as database and this was deployed using docker.
